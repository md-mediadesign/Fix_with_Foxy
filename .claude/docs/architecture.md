# Projekt Architektur

Dieses Projekt ist eine Next.js Plattform für Handwerker- und Serviceangebote.

## Haupttechnologien

- Next.js
- TypeScript
- Prisma
- PostgreSQL (Railway)
- Tailwind CSS

## Wichtige Ordner

src/  
→ Hauptlogik der Anwendung

src/app/  
→ Routing und Seiten

src/components/  
→ Wiederverwendbare UI-Komponenten

prisma/  
→ Datenbankmodell und Seeds

public/  
→ Statische Dateien

media/  
→ Bilder und Assets

## Ziel der Plattform

Eine Plattform ähnlich wie Werkspot, auf der:

- Nutzer Handwerker finden
- Anbieter Profile erstellen
- Angebote vermittelt werden
