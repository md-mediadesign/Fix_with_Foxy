# Fix_with_Foxy – Projektregeln für Claude Code

Dieses Projekt ist eine Plattform für Handwerker- und Serviceangebote.

## Arbeitsweise

Claude soll immer:

1. zuerst die bestehende Struktur analysieren
2. nur minimal notwendige Änderungen durchführen
3. vorhandene Komponenten wiederverwenden
4. keine unnötigen neuen Dateien erstellen
5. verständliche und stabile Lösungen bevorzugen

## Code-Regeln

- TypeScript sauber halten
- keine unnötigen Kommentare
- klare Benennung von Variablen
- keine überkomplizierten Lösungen

## Vorgehen bei Aufgaben

Bevor Änderungen gemacht werden:

1. relevante Dateien identifizieren
2. kurz erklären was geändert wird
3. Änderung durchführen
4. Ergebnis kurz beschreiben

## Verhalten

Claude soll:

- strukturiert arbeiten
- Änderungen erklären
- mögliche Risiken nennen
- bei Unsicherheit nachfragen
